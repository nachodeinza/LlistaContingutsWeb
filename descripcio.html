Descripció i objectius del projecte

Descripció de l'Aplicació: LlistaContinguts

La nostra aplicació és un gestor de llistes de contingut dissenyat per permetre als usuaris crear, emmagatzemar i gestionar el seu propi conjunt de continguts de manera eficient i personalitzada.
Amb un enfocament en la facilitat dús i la funcionalitat robusta, el nostre sistema ofereix una plataforma integral per organitzar i valorar una varietat de contingut segons les preferències de cada usuari.

Característiques Principals

  • Registre i Inici de Sessió d'Usuaris: L'aplicació permet als usuaris registrar-se amb un compte únic i després iniciar sessió per accedir a les llistes de contingut personalitzades. Això s'aconsegueix mitjançant un sistema segur d'autenticació d'usuaris que emmagatzema la informació en una base de dades SQL per garantir la privadesa i seguretat de les dades.
  • Gestió de llistes de contingut: Un cop dins de l'aplicació, cada usuari té la capacitat de crear i mantenir la seva pròpia llista de contingut. Això inclou la possibilitat d'afegir nous elements de contingut, eliminar elements existents i fer modificacions segons calgui. Cada usuari té un identificador únic a la base de dades SQL, cosa que garanteix que la llista de contingut estigui completament separada i privada de la d'altres usuaris.
  • Creació i Eliminació de Contingut: Els usuaris tenen la capacitat de crear nous elements de contingut dins de la llista personalitzada. Això pot incloure una varietat de tipus de contingut, com ara articles, vídeos, imatges, enllaços web, etc. A més, els usuaris poden eliminar contingut de la llista si ja no és rellevan desitjat.
  • Valoració de Continguts: Una funció addicional proporcionada per la nostra aplicació és la capacitat que els usuaris afegeixin valoracions als continguts que han creat. Això permet que altres usuaris vegin la qualitat i la rellevància dun contingut en particular dins de la plataforma. Les valoracions poden ajudar a destacar els millors continguts i millorar lexperiència general de navegació i descobriment per a tots els usuaris.

    • Objectius del projecte

  • Objectiu 1: Desenvolupar una Aplicació Java amb Interfície JavaFX: El primer objectiu daquest projecte és crear una aplicació utilitzant el llenguatge de programació Java i dissenyar la vostra interfície gràfica utilitzant JavaFX. Això implica la creació de finestres, botons, camps de entrada de text i altres elements d'interfície d'usuari necessaris per proporcionar una experiència interactiva i amigable per a lusuari.
  • Objectiu 2: Connectar l'Aplicació a una base de dades MySQL: El segon objectiu se centra a establir una connexió entre l'aplicació desenvolupada a Java i una base de dades MySQL. Aquesta connexió permetrà a l'aplicació emmagatzemar i recuperar dades de la base de dades. que proporcionarà una funcionalitat més avançada i personalitzada per als usuaris. S'implementaran consultes SQL per interactuar amb la base de dades de manera eficient i segura.
  • Objectiu 3: Crear una aplicació per emmagatzemar contingut per a cada usuari registrat a l'app: El tercer objectiu consisteix a desenvolupar una funcionalitat per emmagatzemar contingut específic per a cada usuari registrat a l'aplicació. Això pot incloure perfils d'usuari, preferències, historials de activitat o altra informació rellevant que sigui única per a cada usuari. Es garantirà que aquestes operacions es realitzen de manera segura i que es manegen adequadament els errors i excepcions.
